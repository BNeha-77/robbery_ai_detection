# Predicting Robbery Behavior Using Indoor Security Camera AI

A Django web application integrating AI models to predict suspicious behavior from indoor security camera data.

## Features

- User & Admin authentication
- Dataset upload and preprocessing
- Run multiple ML algorithms and view results
- Admin can manage users
- Graphical reports

## Setup

1. Install requirements: `pip install django`
2. Run migrations: `python manage.py migrate`
3. Create superuser: `python manage.py createsuperuser`
4. Run server: `python manage.py runserver`
